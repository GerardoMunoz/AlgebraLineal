Cada ecuación lineal representa una recta. Usualmente dos rectas se intersecan en un punto. Ahora veremos cómo obtener dicho punto. 
Sistema de dos ecuaciones lineales 
Encontrar un vector que al ser multiplicado por la matriz  ImgStrA $e10 dé como resultado un vector conocido  ImgStrw 
UnaFigura(350,300) -1x + 2y=2
 1x + 3y=1
 [[-1, 2, 2], [1, 3, 1]]
 
Al sumar las dos ecuaciones se elimina la variable x 
UnaFigura(350,300) -1x + 2y=2
 0x + 5y=3
 [[-1, 2, 2], [0, 5, 3]]
 
Al sumar cinco veces la primera ecuación con dos veces la segunda ecuaciones se elimina la variable y 
UnaFigura(350,300) -5x + 0y=4
 0x + 5y=3
 [[-5, 0, 4], [0, 5, 3]]
 
De esta forma se pueden despejar las variables
